Floor and ceil #

Summary #

We define floor, ceil, nat_floorand nat_ceil functions on linear ordered rings.

Main Definitions #

  • floor_ring: Linear ordered ring with a floor function.
  • floor x: Greatest integer z such that z ≤ x.
  • ceil x: Least integer z such that x ≤ z.
  • fract x: Fractional part of x, defined as x - floor x.
  • nat_floor x: Greatest natural n such that n ≤ x. Defined as 0 if x < 0.
  • nat_ceil x: Least natural n such that x ≤ n.

Notations #

  • ⌊x⌋ is floor x.
  • ⌈x⌉ is ceil x.

Tags #

rounding, floor, ceil

Type u_2
  • floor : α → ℤ
  • le_floor : ∀ (z : ℤ) (x : α), z ≤ floor_ring.floor x ↔ ↑z ≤ x

A floor_ring is a linear ordered ring over α with a function floor : α → ℤ satisfying ∀ (z : ℤ) (x : α), z ≤ floor x ↔ (z : α) ≤ x).
